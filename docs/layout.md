# 布局相关
导入
````js
  import { Layout } from '@dtinsight/dt-utils';
````
## pageWidth
获取页面宽度
```js
  Layout.pageWidth()
```
## pageHeight
获取页面高度
```js
  Layout.pageHeight()
```
> 树形布局计算

 根据父节点，计算当前节点的坐标
**参数**
 ````js
 /**
 *
 *根据父节点，计算当前节点的坐标
 * @export
 * @param {*} relativeNode
 * @param {*} node
 * @returns
 */
 Layout.getGeoByRelativeNode (relativeNode: any, node: any);
 ````
